## Getting Started

```bash
# Clone this template
git clone https://github.com/jideabdqudus/my-next-template.git

# Install dependencies
npm install

# Start development server (runs on port 3005)
npm run dev
```

Visit [http://localhost:3005](http://localhost:3005) to see your application.
